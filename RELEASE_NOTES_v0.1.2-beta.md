# Release Notes — v0.1.2-beta

## Summary
v0.1.2-beta is the first beta-quality internal release of LFC Platform.

It provides a working convention operations backbone with attendee QR badge flow, staff check-in, admin analytics, and public API groundwork for WordPress integration.

## Highlights
- real check-in timestamps
- live arrivals feed
- lane analytics
- surge detection
- admin DB viewer
- attendee My Tickets
- attendee My Badge full-screen QR display
- QR PNG rendering
- public event info endpoint
- public ticket tier listing endpoint
- signup/login stability improvements

## Intended Use
- internal admin/staff testing
- attendee flow testing
- website integration groundwork
- WordPress API handoff

## Not Yet Final
- real payment integration
- full production purchase lifecycle
- final public website integration polish
- final scanner workflow polish
